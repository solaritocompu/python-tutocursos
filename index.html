Requisitos para el curso

  1. Tener una computadora y saber usarla también puede ser un celular o una tableta.
  2. Saber usar un editor de texto. No te quejes! Bendito... Quieres programar y no sabes como guardar un archivo de texto. Mejor busca en Internet como hacerlo o coge un curso de computadora antes de seguir este curso. Hay que gatear antes que caminar y caminar antes que correr.
  3. Instalar el Thonny porque incluye el editor y python en un solo instalador. Viene listo para escribir archivos de código sin tener que configurarlo y tiene un área para ejecutar instrucciones en el shell. Web de Thonny para descargarlo e instalarlo

Mostrar texto en el shell

Si abres Thonny se vera lo siguiente abajo:

Python 3.7.9 (bundled)
>>>

Escribe print("Hola mundo") después de >>> y presiona enter. Veras que aparece en el shell Hola mundo o cualquier carácter o numero que introduzcas entre las comillas. La instruccion print significa imprime así que le estamos diciendo a la computadora imprime Hola mundo y ella lo hace. Esta importante instrucción es la primera que se aprende, sirve para mostrar texto y también sirve para mostrar el contenido de variables lo que es muy útil y lo explicare mas adelante en el curso.

Como ejecutar un archivo en Thonny

En Thonny se escribe el código en el área de texto y se ejecutan con la opción del menú correspondiente o presionando f5 si no lo haz guardado saldrá una ventada de guardar archivo ponle el nombre archivo.py y guárdalo. No olvides guardar el archivo antes de ejecutar.

Comentarios

Los comentarios son una forma de explicar como funciona una sección de código, por si necesitas editar tu programa el próximo mes te ayude a recordar como funciona una o varias instrucciones. Los comentarios se escriben así:

#Esto es un comentario y es solo para que alguien lo lea.

Si escribes algo que no sea una instrucción valida sin # delante y ejecutas, nos dará un error y el programa se detendrá. También los comentarios sirven para dejar sin efecto una o varias lineas de código y así el interprete python no las ejecute. Si estas indeciso entre una linea u otra puedes comentar una y probar la otra.

print("Se ejecutara esta linea cuando corra el programa")
#print("Cuando se ejecute el programa esta linea no sera ejecutada y sera tratada como si no estuviera escrita")

Si quieres invalidar muchas lineas o escribir un párrafo de comentario se escribe un comentario múltiple así:

''' Esto es un comentario multilínea,
sirve para documentar una parte
del programa, y se usa comúnmente
para documentar una función.'''

Puedes ponerlo entre tres comillas simples ''' o dobles """.

Variables

Una variable sirve para ponerle un nombre a un dato.

x = 5

Guardamos el dato 5 en la variable x y después se podrán realizar operaciones matematicas con ella

>>> x = 5
>>> print(x + 7) #suma
12
>>> print(x - 2) #resta
3
>>> print(4 * x) #multiplicación
20
>>> print(9 / x) #división
1.8
>>> print(9 // x) #división entera
1
>>> print(x ** 2) #exponente
25
>>> x = x + 1 #Aumentamos uno el valor de x
>>> print(x)
6
>>> x += 1 #Es lo mismo que x = x + 1 abreviado. Hace lo mismo
>>> print(x)
7

Para saber de que tipo es una variable.

>>> type(x) #De que tipo es la variable x?
<class 'int'> #De tipo int, abreviatura del ingles integer que en español es entero.
>>> k = 3.2
>>> type(k)
<class 'float'> #De tipo float

Tipos de variable

a = 5 #integer
b = 7.35 #float
c = "texto escrito" #string
d = [1, 2, 3, 4, 5] #list
