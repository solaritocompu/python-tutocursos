Mostrar texto en python

Si escribes en la consola python y presionas enter Se abrirá lo siguiente:

Python 3.9.7 (tags/v3.9.7:1016ef3, Aug 30 2021, 20:19:38) [MSC v.1929 64 bit (AMD64)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Escribe print("Hola mundo") después de >>> y presiona enter. Veras que aparece en consola Hola mundo o cualquier carácter o numero que introduzcas entre las comillas. Esta importante instrucción es la primera que se aprende, sirve para mostrar texto y también sirve para mostrar el contenido de variables lo que es muy útil y lo explicare mas adelante en el curso.
